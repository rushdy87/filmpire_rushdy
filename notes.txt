npm install @reduxjs/toolkit
The Redux Toolkit package is intended to be the standard way to write Redux logic. It was
originally created to help address three common concerns about Redux:
- "Configuring a Redux store is too complicated"
- "I have to add a lot of packages to get Redux to do anything useful"
- "Redux requires too much boilerplate code"

npm install axios
Axios is a promise-based HTTP Client for node.js and the browser. It is isomorphic (= it can run in
the browser and nodejs with the same codebase). On the server-side it uses the native node.js http
module, while on the client (browser) it uses XMLHttpRequests.

npm install react-redux
React Redux is the official React UI bindings layer for Redux. It lets your React components read
data from a Redux store, and dispatch actions to the store to update state.
